== Description ==

Beautiful admin color theme! If you spend a lot of time with WordPress admin, this is the best choice to make admin beautidul and productive! You suffer from sore eyes? Use this theme! This color scheme developed to prevent suffer from sore eyes. Also it promotes a relax concentration for a long time during your work.

This plugin provides:

* Beautiful light scheme that sore eyes
* Possibility to change admin background
* Possibility to add your own logo in admin menu
* Easy to use and simple to config

== Installation ==

1. Upload eye-candy-theme to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Users -> Your Profile and select Eye Candy from list of color schemes in Admin Color Scheme section.
3. Go to Appearance -> Eye Candy settings and set settings if you want
5. That's all! Just use it and have a fun!
